Platform Capabilities

Tableau Dashboard

Visualisasi interaktif attrition dengan dashboard Tableau. Eksplorasi pola, filter data, dan download semua dashboard sebagai PDF.

Prediksi Assessment

Form assessment lengkap dengan core fields wajib dan advanced options. Input data karyawan untuk mendapatkan prediksi risiko attrition dan rekomendasi aksi HR.

Logistic Regression Model

Model machine learning terlatih untuk prediksi probabilitas attrition. Setiap assessment menghasilkan risk score dan driver utama dari expert system.
